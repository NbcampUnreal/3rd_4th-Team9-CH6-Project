# 🗡️ RemnantSoul

> **Souls-like Action RPG | Unreal Engine 5.6**

---

## 📅 프로젝트 개요

* **제작 기간**: 2025.12.12(금) ~ 2026.01.09(금)
* **플랫폼**: PC (싱글 플레이)
* **장르**: 소울라이크 액션 RPG

---

## 🎮 프로젝트 소개

<p align="center">
  <img width="80%" src="https://github.com/user-attachments/assets/9a9353d1-eba7-4360-873c-2cc86b2464b7" />

**RemnantSoul**은 소울라이크 전투 감성을 기반으로 한 액션 RPG입니다.
실시간 전투 시스템과 **GAS(Gameplay Ability System)** 기반 스킬 구조를 중심으로,
전투의 리스크와 보상을 명확히 설계한 플레이 경험을 제공합니다.

---

| 구분                  | 내용                                      |
| ------------------- | --------------------------------------- |
| **Engine**          | Unreal Engine 5.6                       |

| **Version Control** | Git + GitHub (기능 단위 브랜치 전략)             |
| **Language / Tech** | C++, UMG, GAS (Gameplay Ability System) |
