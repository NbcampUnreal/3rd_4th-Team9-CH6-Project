# RemnantSoul Project


## 프로젝트 소개

<img width="1536" height="1024" alt="Image" src="https://github.com/user-attachments/assets/9a9353d1-eba7-4360-873c-2cc86b2464b7" />

RemnantSoul은 소울라이크 기반의 액션 RPG로 실시간 전투, 캐릭터 성장, GAS 기반 다양한 스킬 사용을 경험할 수 있는 게임입니다.


---

## 개발 환경

* **Engine** : Unreal Engine 5.6
* **IDE** : JetBrains Rider / Visual Studio 2022 (혼합 사용)
* **Version Control** : Git + GitHub (기능별 브랜치 개발)
* **Language/Tech** : C++ / UMG / **GAS(Gameplay Ability System)**

---

## 게임 개요

| 구분    | 내용                                            |
| ----- |-----------------------------------------------|
| 장르    | 소울라이크 액션 RPG                                  |
| 목표    | 전투 및 보스 클리어                                   |
| 핵심 루프 | 일반 몬스터 전투 → 보스 전투 전 정비 → 보스 몬스터 전투 → 클리어 후 정비 |
